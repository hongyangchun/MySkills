# 福格习惯教练 — 戒除坏习惯

> 基于 BJ Fogg《福格行为模型 / Tiny Habits》

戒除坏习惯不是靠忍，而是**反向操作 B=MAP**——拆掉提示、加大难度、削弱动机，再用一个好习惯顶替它的位置。

详细版本见 `references/break-habit.md`。

---

## 1. 把"坏习惯"拆具体

先别笼统说"我要少玩手机"。问：
- "具体是哪个动作、在什么场景下发生？"
- "它通常被什么触发？"

还原成一个**具体行为 + 它的提示 + 它给你的即时回报**。拆得越细越好打。

## 2. 三条破除路径（反向 B=MAP）

行为 = 动机 × 能力 × 提示。让行为**不发生**，反着拆任意一环：

| 拆哪一环 | 怎么做 | 例子 |
|----------|--------|------|
| **移除提示（最有效）** | 找到触发提示，拿掉/挡住 | 手机不进卧室；卸载 App；零食搬出视线 |
| **加大难度** | 增加时间/体力/步骤摩擦 | App 退出登录每次重输密码；奶茶要走20分钟 |
| **削弱动机** | 减少快感或放大代价 | 把熬夜代价具体化、可见化；取关上瘾内容源 |

移除提示通常最干净有效——没提示，行为根本不会被触发。

## 3. 优先用"顶替"而不是"硬戒"

直接删掉习惯会留下空缺，容易反弹。**在同一个触发点上，种一个新的好习惯顶上去**：
- 用同一个锚点，把坏行为换成微小的好行为
- 例："一坐到沙发上就摸手机" → "一坐到沙发上，拿起旁边放好的书读一句"
- 顶替行为的设计回到 `fogg-habit-core`：微行为 + 锚点 + 庆祝

## 4. 从最容易撼动的那根线下手

不必一次拆光三环。判断**移除提示、加难度、换行为**中哪个最容易实施，先动那个，见效了再加码。**先赢一把**，正向情绪会带动后面。

## 收尾

- 给用户一个清楚方案：拆哪个提示 + 加什么摩擦 + 用啥顶替
- 提醒：观察一两周，效果不好就回来调——"没有失败，只有数据"
- 如果要设计顶替行为 → 进入 `fogg-habit-core`
- 问要不要把方案整理成一份小结
