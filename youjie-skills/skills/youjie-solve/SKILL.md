# KSME 问题解决七步法

> 基于《有解》第3-8章，KSME = K(Knowledge)知识 + S(Skill)技能 + M(Motivation)动机 + E(Environment)环境

## 七步法全景图

```
A阶段（定向）              B阶段（分析）              C阶段（行动）
┌─────────────┐         ┌─────────────┐         ┌─────────────┐
│ ① 确定问题   │         │ ③ 明确现状   │         │ ⑥ 制定方案   │
│ ② 梳理关系人 │───────→ │ ④ 明确目标   │───────→ │ ⑦ 拟定行动   │
│             │         │ ⑤ 明确差距   │         │             │
└─────────────┘         └─────────────┘         └─────────────┘
  聚焦方向，有的放矢       避免草率行动             排除干扰，达成目标
```

**引导节奏**：
- 每一步只问 1-2 个核心问题，等用户回答后再推进
- 每步结束时输出该步的"成果摘要"
- 三阶段之间有自然停顿点：A阶段结束确认"方向对了吗"，B阶段结束确认"分析够深了吗"，C阶段结束确认"可以行动了吗"
- 全程7步不一定一次走完——可以分多次，每次聚焦1-2步

**前置条件**：如果用户情绪状态不佳或尚未确立"问题管理者"角色，建议先进入 `youjie-prepare` 模块完成准备。

## 第①步：确定问题

**目标**：从一堆问题中找到"问题之王"——那个影响全局的核心问题。

### 工具 1：问题清单

把所有问题全部写下来，不再让它们在脑子里转。

- 核心价值：停止"过度思考"，把问题外化
- 过度思考的特征：反复想同一个问题但没有任何行动
- 引导："你现在面临的所有问题，一个一个列出来，不用排序，想到什么写什么。"

### 工具 2：价值罗盘

在以下维度给每个问题打分（1-10）：
- **重要性**：这个问题对你有多重要？
- **紧迫性**：这个问题需要多快解决？
- **影响范围**：这个问题影响你生活的多少方面？

目的：从"感觉都很重要"变成"数据说话"。用可用的可视化方式呈现打分结果（如雷达图），或用文字描述对比。

### 工具 3：紧急重要模型

横轴：紧急性（左低右高），纵轴：重要性（下低上高）

四象限：
- **重要且紧急** → 优先处理
- **重要不紧急** → **最该投入的领域**（常被忽视！）
- **紧急不重要** → 尽量委托或减少
- **不重要不紧急** → 放手

警惕陷阱：误把紧急当重要。紧急的事让人焦虑但未必重要，重要的事不紧急但决定长期。

### 工具 4：两难选择框架

当面临"选A还是选B"时：
- 选项本身没有优劣，都在同一价值范畴里
- 关键是"是谁在做选择"——你更在乎的价值是什么
- 依据自己看重的价值做出选择，才是无悔的
- 引导："这两个选项背后，你真正在乎的是什么？"

### 工具 5：有的放矢，有的放手

- 不是所有问题都要解决
- 忽略该忽略的和重视该重视的，同等重要
- 引导："如果只能解决一个问题，你会选哪个？为什么？"

### Step 1 引导流程

```
1.1 列出所有问题（问题清单）
1.2 用价值罗盘给每个问题打分
1.3 放入紧急重要模型
1.4 找到"问题之王"——那个重要且影响全局的
1.5 确认："这就是你要聚焦解决的问题吗？"
```

**成果摘要**：一个问题之王 + 选择理由

## 第②步：梳理关系人

**目标**：找到解决问题的盟友，组建"问题解决项目团队"。

**核心认知**：再有能力也不能单打独斗。推动问题解决的不是权威，而是关系。

### 工具 1：人际生态图

画出你身边所有人际关系：
- 标注每段关系的质量（强/弱/紧张/疏远）
- 发现被错过的资源——那些你忽略的人可能正是关键盟友
- 引导："在这个问题上，谁和你有关？把所有人画出来。"
- 可用可视化方式呈现关系网络图，或用文字描述

### 工具 2：关系人图

对问题之王，更细致地分析关系人：
- 谁是**支持者**？谁是**阻碍者**？谁是**旁观者**？
- 每个关系人的**真实需求**是什么？
- 关系卡在哪里？
- 关键公式：**关系决定结构，结构决定功能**

### 工具 3：双重身份模型

你有两个身份：
- **重要关系人**：你是问题的当事人，有情感投入
- **问题管理者**：你需要跳出来，客观地管理问题

这两个身份不矛盾——先承认自己有情感投入，再以管理者身份行动。

### 工具 4：关系体检

为关键关系做"体检"：
- 这段关系现在是**健康的**还是**危险的**？
- 如果是危险关系，提前预警，不要等它爆炸
- 关系卡在哪了？看见 Ta 真正的需求

### Step 2 引导流程

```
2.1 画出人际生态图
2.2 针对问题之王，画关系人图
2.3 标注每个关系人的角色（支持/阻碍/旁观）和需求
2.4 做关系体检，发现危险关系
2.5 明确你的双重身份
2.6 确认："你的问题解决团队有哪些人？"
```

**成果摘要**：关系人清单 + 每人角色与需求 + 危险关系预警

## 第③步：明确现状

**目标**：用事实而非观点描述现状，让真正的问题浮出水面。

**核心认知**：多数人描述的"现状"其实是"观点"，不是"事实"。

### 工具 1：观点 vs 事实区分法

- **观点**：带有判断、情绪、评价的描述
  - 例："我老板总是针对我"（观点）
- **事实**：可观察、可验证、不带评价的描述
  - 例："上周老板在我的报告上标注了7处修改意见，同事的报告平均3处"（事实）
- 引导："你刚才说的这句话，是观点还是事实？能找到对应的事实吗？"

### 工具 2：警惕高度概括

高度概括是"观点陷阱"的高级形式：
- "他**从来不**关心我" → 事实是什么？有几次？什么情境？
- "我**什么都**做不好" → 具体哪些事做不好？哪些做得还行？
- 高度概括带来的伤害非常大——它把局部问题放大成全局否定
- 信号词：总是、从来不、每次都、什么都、所有人
- 引导："把这句话拆开——具体是什么事？什么时间？什么情境？"

### 工具 3：负面标签剥离

当观点不断重复，就变成了标签。

负面标签的力量非常强大——它为你设计了一个充满负面评价的牢笼。

核心原则：
- 某个**行为**需要矫正 ≠ 你这个**人**不行
- 某项**能力**需要提升 ≠ 你没有**价值**
- 作为问题管理者，任何时候都不否定自己

金句："只要你决定不给自己贴上负面标签，任何人都无法给你贴上。"

引导："你给自己贴了什么标签？这些标签是事实吗？"

### 工具 4：细化与量化（问题X光片）

把问题横向展开，不要只往深处挖。给问题拍一张"X光片"——尽可能量化。

- 不量化 → "我工作效率很低"（模糊，无法行动）
- 量化后 → "本周完成3项任务，平均每项耗时2天，同事平均1天"（清晰，可行动）

引导："如果给这个问题拍一张X光片，数据是什么样的？"

### 工具 5：影响圈

区分两个圈：
- **关注圈**：你在意但无法控制的（别人的态度、经济环境、过去的事）
- **影响圈**：你能直接影响的（你的行为、你的沟通方式、你的选择）

把精力集中在影响圈内。影响圈会随着你的行动逐渐扩大。

引导："在这个问题里，哪些是你能直接改变的？哪些是你控制不了的？"

### Step 3 引导流程

```
3.1 描述现状（先让用户自由说）
3.2 区分观点和事实，找到事实
3.3 拆解高度概括，剥离负面标签
3.4 细化并量化现状（X光片）
3.5 圈定影响圈——你能直接改变的部分
3.6 让真正的问题浮出水面
```

**成果摘要**：用事实描述的、量化的、聚焦影响圈的现状

## 第④步：明确目标

**目标**：找到藏在问题背后的目标，把"不想要的"转化为"想要的"。

**核心认知**：解决问题的真正转机，在于不再和问题硬碰硬，而是绕到它后方，让藏在问题背后的目标重现。

### 工具 1：问题→目标转换

每个问题的背后都藏着一个目标：
- 问题："我总是拖延" → 目标："我要如何高效行动？"
- 问题："团队沟通不畅" → 目标："我要如何建立顺畅的沟通机制？"
- 问题："孩子不爱学习" → 目标："我要如何激发孩子的学习兴趣？"

暂停思考"为什么"，转而思考"要什么"。

引导："如果你不再有这个问题，你想要的是什么状态？"

### 工具 2：SMART 目标原则

- **S**pecific：具体的（不是"变得更好"，而是"每月读2本书"）
- **M**easurable：可衡量的（有明确指标）
- **A**chievable：可实现的（跳一跳够得着）
- **R**elevant：相关的（与你的核心价值一致）
- **T**ime-bound：有时限的（有截止日期）

引导："你的目标符合SMART吗？哪一项需要调整？"

### 工具 3：蓝莓枝修剪

目标不是越多越好——人的时间、精力、资源都是有限的。

大胆剪掉不重要的目标，像修剪蓝莓枝一样。聚焦：一个时期内只追求1-3个核心目标。

引导："如果只能保留一个目标，你保留哪个？"

### 工具 4：目标转向

有时你追求的不是真正的目标，而是达成目标的某条路径：
- 例："被提拔"是路径，"职业成长"才是目标——不被提拔也能成长
- 当路径被堵时，检查：这是路径还是目标？如果是路径，换一条路

引导："你追求的这个目标，是终点还是某条路？"

### 工具 5：目标平衡

不能让目标倒向一边——工作、家庭、健康、成长需要平衡。

引导："你的目标覆盖了生活的哪些方面？有没有被忽视的重要方面？"

### 工具 6：愿望强度评估

1-10分：你有多想实现这个目标？
- 如果 ≤ 6分：可能不是真正的目标，重新审视
- 如果 ≥ 8分：有足够的内驱力

### Step 4 引导流程

```
4.1 从问题中提取目标（问题→目标转换）
4.2 用SMART原则检验目标
4.3 修剪目标（蓝莓枝）
4.4 检查是否需要目标转向
4.5 检查目标平衡
4.6 评估愿望强度
```

**成果摘要**：一个清晰的、SMART的、有内驱力的目标

## 第⑤步：明确差距与代价

**目标**：量化现状与目标之间的差距，以及差距不被弥补的代价，产生行动推力。

**核心认知**：差距（Δ）是你最重要的推力——知道自己为何而战。

### 工具 1：Δ 差距计算

现状值 vs 目标值，量化差距：
- 例：现状月收入1万，目标月收入1.5万，Δ = 5000
- 例：现状每周运动0次，目标每周3次，Δ = 3次
- 例：现状团队流失率20%，目标10%，Δ = 10个百分点

引导："现状和目标之间的差距，用数字表达是多少？"

### 工具 2：代价量化

如果差距不被弥补，会付出什么代价？

- **短期代价**：1个月后会怎样？
- **中期代价**：1年后会怎样？
- **长期代价**：5年后会怎样？

目的：不是制造焦虑，而是让"不行动"的代价显性化。

引导："如果这个问题一直不解决，1年后你的生活会怎样？5年后呢？"

### 工具 3：他律 vs 自律

- **他律**：靠外部压力推动（被催才动）——不可持续
- **自律**：靠内在驱动行动（自己想动）——可持续
- 差距和代价的量化，帮助从"他律"转向"自律"
- 当你真正看清代价，动力会从内而生

引导："你现在是靠外部压力行动，还是靠内在驱动？"

### Step 5 引导流程

```
5.1 量化现状值和目标值
5.2 计算差距Δ
5.3 量化代价（短期/中期/长期）
5.4 评估动力来源（他律 vs 自律）
```

**成果摘要**：清晰的差距 + 不行动的代价 + 行动的内驱力

## 第⑥步：制定解决方案

**目标**：从 K、S、M、E 四个维度排除干扰，用创意和共识制定方案。

**核心认知**：你的潜能足够，只需排除干扰。

### 工具 1：绩效公式 P = p - i

- **P**（Performance）= 实际表现
- **p**（potential）= 潜能
- **i**（interference）= 干扰

核心洞察：提升表现不一定要增加潜能，减少干扰同样有效——而且通常更容易。

引导："是什么在干扰你？如果排除这些干扰，你的表现会怎样？"

### 工具 2：KSME 四维度排除干扰

| 维度 | 含义 | 干扰来源 | 解决方向 |
|------|------|---------|---------|
| **K**（Knowledge，知识） | 你缺少什么知识/信息？ | 不知道怎么做 | 学习、请教、查找资料 |
| **S**（Skill，技能） | 你缺少什么技能？ | 知道但做不到 | 练习、培训、找人协助 |
| **M**（Motivation，动机） | 你的动力够吗？ | 不想做、害怕做、觉得没意义 | 重新连接目标、调整心态、寻找意义 |
| **E**（Environment，环境） | 环境支持你吗？ | 工具不够、时间不够、空间不对、人际关系消耗 | 改变环境、争取资源、调整时间安排 |

**排序原则**：M（动机）排第一——动机不对，其他三个都使不上劲。

引导："从K、S、M、E四个方面看，你的干扰分别是什么？"

### 工具 3：头脑风暴

- **发散阶段**：只产出不评判，越多越好
- **收敛阶段**：筛选可行的方案
- 规则：先发散再收敛，不要边想边否定

### 工具 4：六顶思考帽

用六种视角系统评估方案：

| 帽子颜色 | 视角 | 关注问题 |
|---------|------|---------|
| ⚪ 白帽 | 事实 | 有哪些数据和事实？ |
| 🔴 红帽 | 感受 | 直觉和情绪怎么说？ |
| ⚫ 黑帽 | 谨慎 | 风险和问题是什么？ |
| 🟡 黄帽 | 乐观 | 好处和机会是什么？ |
| 🟢 绿帽 | 创意 | 还有没有新的可能？ |
| 🔵 蓝帽 | 管理 | 总结和下一步是什么？ |

引导："我们戴上不同颜色的帽子来看这个方案——先戴白帽，事实是什么？"

### 工具 5：真心欣赏

水面之下最容易被忽视的难题：不欣赏。

- 为什么表达欣赏比表达批评更难？批评指向问题，欣赏需要看见人的价值。我们习惯了"挑错"而非"看见好"。
- 欣赏是"零成本"的强大积分项：真心欣赏自己和他人 → 打开合作的大门
- 欣赏不是奉承，是看见真实的价值

引导："在这个方案里，每个人的贡献是什么？你真心欣赏谁？"

### 工具 6：信任与托付

- 相信而敢于托付 → 来到良性循环的入口
- 不信任的代价：所有事都自己做，累死且做不好
- 引导："在这个方案里，哪些部分可以托付给谁？你信任他们吗？"

### 工具 7：投入聆听

- 最稀缺的能力：请别人多说，让自己多听
- 聆听不是等对方说完再反驳，而是真正理解对方
- 引导："你有多久没有真正聆听过身边的人了？"

### Step 6 引导流程

```
6.1 用 P=p-i 识别干扰
6.2 从 K/S/M/E 四维度分析干扰来源
6.3 针对每个维度，头脑风暴解决方案
6.4 用六顶思考帽评估方案
6.5 处理暗礁：欣赏 → 信任 → 聆听
6.6 达成共识："这是'我们的方案'吗？"
```

**成果摘要**：一个综合的、有创意的、有共识的解决方案

---

## 第⑦步：拟定行动计划

**目标**：把方案变成可执行的行动，用 PDCA 确保落地。

**核心认知**：谁想解决问题，谁先改变——首先改变的人最有力量。

### 工具 1：谁先改变

- 改变的原则：不是"谁有错谁先改"，而是"谁想解决问题谁先改"
- 首先改变的人最有力量——你改变自己的同时也会带动他人改变
- 引导："在这个计划里，你愿意先改变什么？"

### 工具 2：从小开始

- 用小的尝试，成就轰轰烈烈的改变
- 不要一上来就做大的——大的改变阻力大、风险高
- 找到"如果可以做一点点，那会是什么？"
- 引导："如果只做一件最小的事，那会是什么？"

### 工具 3：行动计划表

每个行动项明确：
- **做什么**（具体行动）
- **谁来做**（责任人）
- **什么时候做**（时间节点）
- **怎么做**（具体步骤）
- **做到什么程度**（验收标准）

引导："把方案拆成具体的行动项，每项都要可执行、可验收。"

### 工具 4：PDCA 循环

- **P**lan（计划）：制定行动计划
- **D**o（执行）：执行计划
- **C**heck（检查）：检查执行效果
- **A**ct（调整）：根据检查结果调整计划

核心价值：让行动计划具有弹性——不是"一次定终身"，而是"持续优化"。

引导："执行一段时间后，我们怎么检查效果？发现偏差怎么调整？"

### 工具 5：目标回归线

你的行动在"目标回归线"上吗？

每个行动都应该指向目标——偏离目标回归线的行动，再忙也是白忙。

引导："这个行动和你的目标之间是什么关系？是在靠近目标吗？"

### Step 7 引导流程

```
7.1 确认"谁先改变"
7.2 找到最小行动起点
7.3 制定行动计划表（5W）
7.4 设定 PDCA 检查节点
7.5 检查每个行动是否在目标回归线上
```

**成果摘要**：一张可执行、可检查、可调整的行动计划表

---

## 完整引导流程总结

```
准备 → ①确定问题 → ②梳理关系人 → ③明确现状 → ④明确目标 → ⑤明确差距 → ⑥制定方案 → ⑦拟定行动
         A阶段（定向）           B阶段（分析）                    C阶段（行动）
```

### 交互要点

- **不要急于跳到解决方案**——B阶段（分析）是最容易被跳过但最关键的
- 每步的工具不是全部都用——根据问题性质选择合适的工具
- 工具是手段不是目的——如果用户已经自然到达了某步的结论，不必强行走工具流程
- 暗线贯穿：在引导过程中自然融入思维转换和理念（详见 `youjie-mindset` 模块）
- 用可用的可视化方式呈现关键分析（如紧急重要四象限、关系人网络图、差距对比图等），或用文字描述

## ⚠️ 边界与局限

KSME 七步法是一个强大的问题解决框架，但它并非万能。以下场景需要注意其局限性：

### 1. 存在性危机场景

对于丧亲、意义缺失、身份危机等存在性问题，Δ公式（|现状-目标|）过于工具化。将"悲痛"定义为需要被缩小的"差距"，可能对当事人造成二次伤害。这类问题需要的不是"解决"，而是陪伴、见证和意义的重新建构。

### 2. 心理健康边界

对于有严重心理疾病的人（如重度抑郁、创伤后应激障碍等），七步法的结构化引导可能不够，甚至可能因过早聚焦"目标"和"行动"而增加压力。这类情况需要转介专业心理干预。

### 3. 潜能假设的限制

P=p-i 公式假设"每个人的潜能都是充足的"，这在大多数情境下成立。但某些情境确实存在知识、能力或资源的硬性缺口——比如极端贫困、严重技能欠缺、信息完全封闭等。此时除了排除干扰，还需要先补足基础能力。

### 4. 安全与伦理边界

"谁的问题谁做主"是 KSME 的核心原则，但在以下情况不适用：
- 涉及人身安全的问题（如家庭暴力）
- 涉及法律边界的问题（如违法行为）
- 涉及伦理底线的问题（如严重伤害他人利益）

这些场景下，专业介入和强制干预优先于"问题管理"。

### 5. 领域适配性

KSME 源于绩效咨询（Performance Consulting）传统，对以下问题的适配性有限：
- 纯粹的艺术创作问题（需要的是灵感，不是排除干扰）
- 深度的自我探索问题（需要的不是目标导向，而是开放觉察）
- 系统性结构压迫问题（个人层面的 KSME 不足以改变制度性不公）

在这些场景下，KSME 可以作为辅助视角，但不适合作为主导框架。建议结合 `systems-thinking` 等其他工具协同使用。
